# screen_description

## ディレクトリ
V-Effect\lib\screens

## ファイル名と説明
- auth_wrapper.dart: ログイン画面の裏処理
- camera_screen.dart: カメラ起動→写真撮影→トリミング→ひとこと追加までの画面
- edit_profile_screen.dart: プロフィール設定画面
- email_verification_screen.dart: メール認証画面
- follow_list_screen.dart: フォローリスト画面
- forgot_password_screen.dart: パスワード変更画面（リセットメール送る前）
- friends_screen.dart: 謎（古いゴミっぽい）
- hero_tasks_screen.dart: 自分のヒーロータスクカードの画面（ナビゲーションバーの真ん中）
- home_screen.dart: 友達の投稿を見る画面（ナビゲーションバー左）
- initial_friend_screen.dart: 初回のフォローを送る画面
- login_screen.dart: ログイン画面
- main_shell.dart: ナビゲーションバー（？）
- notifications_screen.dart: 通知画面
- pending_requests_screen.dart: フォロー申請画面
- profile_screen.dart: プロフィール画面（ナビゲーションバーの右）
- profile_setup_screen.dart: サインアップ時のプロフィール設定画面
- register_screen.dart: 新規登録画面
- reset_password_screen.dart: パスワード変更画面（リセットメール送った後）
- search_screen.dart: ユーザー検索画面
- settings_screen.dart: 設定画面（その他の設定）
- task_setup_screen.dart: 初期ヒーロータスク設定画面（今使われてなさそう→task_template_screen.dartに吸収されている）
- task_template_screen.dart: 初期ヒーロータスク設定画面（まずはこれをやってみようの画面）
- user_profile_screen.dart: 他ユーザーのプロフィール画面
- weakly_review_screen.dart: ウィークリーレビュー画面
