# 日々の開発ワークフロー

> **このファイルは何？**
> Antigravity に `/daily-dev` のようにワークフロー名を伝えると、
> ここに書かれた手順を自動で実行してくれます。

## 手順

### 1. 最新のコードを取得（Pull）

まず、パートナーが変更した最新のコードを自分のパソコンに取り込みます。

```bash
git pull origin main
```

> もしエラーが出た場合は、Antigravity に「Pull でエラーが出たので解決して」と伝えてください。

### 2. 開発作業

Antigravity に作りたい機能やデザインを指示して、コードを書いてもらいます。

**話しかけ方の例：**
- 「ログイン画面を作って」
- 「ボタンを押したら次の画面に遷移するようにして」
- 「このエラーを直して」

### 3. 変更をまとめて保存（Commit）

キリのいいところまでできたら、変更内容を保存します。

```bash
git add -A
git commit -m "変更内容の説明を書く"
```

> Antigravity に「今の変更をコミットして」と言えば、適切なメッセージ付きで自動実行してくれます。

### 4. パートナーと共有（Push）

保存した変更を GitHub にアップロードして、パートナーにも見えるようにします。

```bash
git push origin main
```

> Antigravity に「Push して」と言えば自動実行してくれます。

## まとめ（1行で言うと）

**Pull → 作業 → Commit → Push** を繰り返すだけです！
分からないことがあれば Antigravity に何でも聞いてください。
